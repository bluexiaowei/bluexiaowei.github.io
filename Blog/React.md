# ReactJS

> 用于构建用户界面的 JavaScript 库。
> 特点：声明式、组件化。

## 前言

## 术语

| 术语              | 解释                                 |
| :---------------- | :----------------------------------- |
| JSX               | 是一个 JavaScript 语法扩展，语法糖。 |
| Components        | 可复用的小的代码片段                 |
| props             | 父组件向下传递给子组件的数据         |
| state             | 组件自身状态                         |
| Lifecycle Methods | 组件从挂在到卸载的生命周期的方法     |
