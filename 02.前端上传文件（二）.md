### 项目

#### 项目快速搭建

使用 UmiJS 应用框架快速搭建项目，省去好多好多麻烦事。
项目创建的时候请依次选择 `app` => `Y` => `antd`

```bash
# 新建应用目录
$ mkdir myapp && cd myapp

# 根据目录创建项目
$ yarn create umi
```
